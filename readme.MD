# RD-autopeças
Projeto de um ERP full stack para uma loja de autopeças, desenvolvido com Java, Spring Boot e React, seguindo práticas utilizadas em projetos reais.

## Tecnologias

- **Backend:** Java 21, Spring Boot, Hibernate/JPA, Lombok, Flyway
- **Banco de dados:** PostgreSQL

## Swagger
Swagger disponível em `http://localhost:8080/swagger` com a aplicação rodando.

## Regras de negócio

- Um cliente e funcionario pode ter o mesmo user,
